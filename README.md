📊 Retail Sales Dashboard (Power BI)

🔍 Project Overview

This project analyzes retail sales data using Power BI to uncover insights about sales performance, customer behavior, and product trends.

📈 Features
KPI Cards: Sales, Profit, Orders, Quantity
Sales Trend Analysis (Line Chart)
Sales by Category (Bar Chart)
Payment Mode Analysis (Donut Chart)
Geographic Sales Distribution (Map)
Product Performance Analysis
🛠 Tools Used
Power BI
DAX
Excel / CSV

💡 Key Insights
Electronics category generates highest sales
UPI is the most used payment method
Some products have high sales but low profit
